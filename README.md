# At a Glance

`SwiftyPredictor` simplifies work with [Yandex.Predictor](https://tech.yandex.ru/predictor) service in iOS.

## How To Get Started

- Copy content of `Source` folder to your project.

or

- Use `SwiftyPredictor` cocoapod.

## Requirements

* iOS 9.0 and later
* Xcode 8 and later
